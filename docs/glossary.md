# Glosario v0.1

## Funciones PdM
- fault_detection: detección de fallas/anomalías
- fault_identification: identificación/clasificación de fallas
- degradation_assessment: evaluación de degradación/health index
- RUL_prognosis: pronóstico de vida remanente (RUL)
- maintenance_scheduling: planificación/optimización de mantenimiento

## Sensores (sinónimos)
- vibration (vibraciones, acelerómetros)
- current (corriente)
- voltage (voltaje, tensión)
- temperature (temperatura)
- pressure (presión)
- acoustic (acústico)
- oil_analysis (aceite, tribología)
- images (visión, cámaras, termografía)

## Métodos
- RF (random forest), XGB (xgboost), SVM, KNN
- LSTM, CNN, Transformer, Autoencoder
- PhysicsInformed, Hybrid

## Métricas
- F1, AUC, Accuracy, Precision, Recall
- MAE, RMSE, MAPE
- RUL error (MAE_RUL, RMSE_RUL)
